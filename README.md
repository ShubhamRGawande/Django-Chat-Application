# 💬 Chatify - One-to-One Chat Application

Chatify is a **Django-based one-to-one chat application** that enables users to register, log in, and chat privately in a secure environment. Designed with a modern, WhatsApp-inspired user interface using Bootstrap, this project demonstrates core web development skills, including backend logic, user authentication, and dynamic UI rendering.  

🔨 *Currently under development – Hosting planned soon!*

---

## 🚀 Features

- ✅ **User Authentication**
  - Secure registration, login, and logout system.
  - Password encryption using Django’s built-in authentication system.
- 💬 **One-to-One Chat**
  - Users can send and receive private messages with other registered users.
- 🟢 **Online Status**
  - Shows which users are active.
- 🖥️ **Modern UI**
  - Clean and responsive Bootstrap design for mobile and desktop.
- 🛠️ **Admin Panel**
  - Manage users and conversations with Django admin.
- 🌱 **Future Enhancements**
  - Real-time messaging using Django Channels (WebSocket support).
  - File and media sharing.
  - Group chats and emojis.
  - Hosting on **Render** or **Heroku** for public access.
